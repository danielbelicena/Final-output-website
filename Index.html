About This Website

New Gen Gaming is a student-made website designed to showcase popular video games in a fun, interactive way. You can explore games, view details, and enjoy a clean browsing experience.

This project is built using HTML, CSS, and Bootstrap. It includes clickable game cards with modal popups, a responsive layout for any device, and smooth navigation between sections. The goal is to create a website that looks professional while staying user-friendly.

The website is perfect for practicing web development skills, learning how to structure pages, and experimenting with design. All images and information are for educational purposes, and this site is not affiliated with any game companies.

We hope this website inspires others to create their own projects and explore coding in a creative, fun way!
